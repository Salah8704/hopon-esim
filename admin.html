⚠ Correction Maroc Illimité 7j :
L'ancien prix "19,90€" publié sur le site n'avait aucune source API identifiable (supplier_price=null, pas de source_product_id). Il s'agissait d'un prix inventé, incompatible avec les règles de publication. Ce produit est maintenant en api_error et retiré du site. Il sera réactivé uniquement après sync OCS réelle + définition du public_price par un admin + validation.
→ Contacter Operator D pour obtenir le pricing officiel de ce forfait.

Règles de publication

1. Prix source exclusif : GET /ocs/catalog/api/cos/{cosRef}/products/{productId}
2. supplier_price ≠ null et ≠ 0 obligatoire avant tout affichage
3. public_price saisi manuellement par admin (jamais auto-calculé sans règle définie)
4. is_published = true uniquement si status === 'validated'
5. Toute modification → log horodaté avec admin_id
6. WooCommerce reçoit les prix via PUT /wc/v3/products/{id} après validation uniquement
7. Aucune extrapolation de prix à partir d'une autre durée ou d'un autre pays
